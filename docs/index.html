Chat Application Backend API (0.1.0)

Download OpenAPI specification:

REST API for a chat application supporting user authentication, channels, and messaging.

Authentication

Most endpoints require authentication via a session token obtained through the /auth/login endpoint. Include the token in the Authorization header as a Bearer token.
