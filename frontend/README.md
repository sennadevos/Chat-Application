# Chat Application Frontend

A modern, production-grade chat frontend built with Vanilla JavaScript (ES6 Modules) and Tailwind CSS.

## Prerequisites

- Node.js & npm (for building CSS)
- A modern browser (Chrome, Firefox, Edge, Safari)

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Start the Tailwind CSS watcher (for development):
    ```bash
    npm run dev
    ```

3.  Open `index.html` in your browser.
    -   **Note:** Because this project uses ES6 Modules and Import Maps, you likely need to serve it via a local web server to avoid CORS issues with `file://` protocol (though some browsers might be strict).
    -   You can use VS Code's **Live Server** extension or python:
        ```bash
        python3 -m http.server
        ```
        Then visit `http://localhost:8000`.

## Project Structure

-   `src/`: Source CSS files.
-   `js/`: JavaScript modules.
    -   `router.js`: Handles navigation.
    -   `auth.js`: Authentication logic & UI.
    -   `chat.js`: Main chat interface logic.
    -   `api.js`: Fetch wrapper.
    -   `websocket.js`: Real-time connection.
    -   `ui.js`: Shared UI components.
-   `dist/`: Compiled CSS (generated by npm run dev/build).
-   `index.html`: Entry point.

## Configuration

-   **API URL**: Configured in `js/api.js` (default: `http://localhost:8080/api`).
-   **WebSocket URL**: Configured in `js/websocket.js` (default: `ws://localhost:8080/api/ws`).
-   **Tailwind**: Configured in `tailwind.config.js` using the specific project color palette.
